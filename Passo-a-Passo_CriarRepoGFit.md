# Passo-a-passo para crir um repositório GitHub

Use:

```bash
git clone <URL_DO_REPOSITORIO>
```
Ou faça tudo do zero
```bash
cd /Diretorio-do-Projeto
git init 
# Reinitialized existing Git repository in D:/Projetos/kcpocketPhp/.git/
git add README.md
git commit -m "Meu primeiro commit"

git remote add origin <URL_DO_SEU_REPOSITORIO>
# git remote add origin https://github.com/nosredna33/kcpocketPhp.git
git commit -m "first commit"

git branch -M main
git push -u origin main
git branch -M main

```
